# Contributing to Juninmd Profile

Thank you for your interest in contributing to my profile repository! This repository is primarily for my GitHub profile README and related automations.

## How to Contribute

1.  **Fork the repository**.
2.  **Create a branch** for your feature or fix.
3.  **Make your changes**.
4.  **Test your changes** (e.g., if you modify workflows, ensure they are valid YAML).
5.  **Submit a Pull Request**.

## Guidelines

-   **Workflows**: If you modify GitHub Actions workflows, please ensure they are valid.
-   **Profile Content**: Changes to `README.md` should maintain the aesthetic and personal nature of the profile.
-   **Scripts**: If you add scripts, please document them.

## Reporting Issues

If you find a bug or have a suggestion, please open an issue using the provided templates.
